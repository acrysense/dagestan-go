Политика оператора в отношении обработки персональных данных

Политика обработки персональных данных содержит основные положения, обязательства оператора и права субъекта персональных данных в соответствии с законодательством Российской Федерации. Она определяет, как ООО «Лазурный берег» осуществляет сбор, хранение, использование, передачу и защиту персональной информации пользователей.


Основные положения:

  1. Политика действует в соответствии с Законом о персональных данных (152-ФЗ).
  2. Оператор несет ответственность за защиту прав субъектов данных.
  3. Персональные данные включают всю информацию, которая может использоваться для идентификации лица.

Цели обработки данных:

  1. Заключение и исполнение договоров.
  2. Информирование пользователей через различные каналы.

Права субъекта данных:

  1. Получение информации о факте обработки данных.
  2. Возможность запроса на уточнение, блокировку или удаление данных.
  3. Право отозвать согласие на обработку данных.

Обязанности оператора:

  • Защита данных от несанкционированного доступа.
  • Хранение данных в России.
  • Предоставление информации по запросу субъекта.

Заключение: Пользователь может обратиться к оператору для разъяснения вопросов по обработке персональных данных через электронную почту info@lazber.ru.
